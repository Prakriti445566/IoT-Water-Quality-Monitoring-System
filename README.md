# IoT-Water-Quality-Monitoring-System
ESP32-based IoT system for real-time monitoring of pH, Turbidity, TDS, and Temperature using Blynk IoT with cloud-based visualization and alerts.
##  Overview
This project is an ESP32-based IoT system that monitors water quality parameters such as pH, Turbidity, Total Dissolved Solids (TDS), and Temperature in real time. The sensor data is transmitted to the Blynk IoT platform for remote monitoring and instant alerts.

##  Features
- Real-time monitoring of water quality
- pH measurement
- Turbidity measurement
- TDS measurement
- Temperature monitoring
- Live data visualization using Blynk IoT
- Low-cost and portable design

##  Hardware Used
- ESP32 Development Board
- pH Sensor
- TDS Sensor
- Turbidity Sensor
- DS18B20 Temperature Sensor
- 16×2 LCD Display
- Buzzer
- Jumper Wires
- Breadboard

## Software Used
- Arduino IDE
- Blynk IoT
- ESP32 Board Package

## Libraries
- Blynk
- OneWire
- DallasTemperature
- LiquidCrystal_I2C

## Working
The ESP32 collects data from multiple water quality sensors. The readings are processed and sent to the Blynk IoT cloud. Users can monitor water quality remotely through the Blynk mobile application.

## Results
- Real-time monitoring of pH, Turbidity, TDS, and Temperature
- Instant cloud updates through Blynk IoT
- Continuous monitoring for improved water quality assessment
